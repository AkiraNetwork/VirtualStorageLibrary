    Class PowerShellWildcardMatcher

    Implements wildcard matching based on PowerShell's wildcard patterns. Provides pattern matching with support for specific escape characters.

    Remarks

    If a wildcard character is escaped using a backtick (`), the character is treated as a literal. The escape character itself is represented by the backtick (`), and the character following it is the one being escaped.
